# ToDo
My application to maintain ToDO list

Download and unzip TodosApp.zip file

on command line goto folder location which has this todos.war file

execute below command

java -jar todos.war


then below URL on web browser

http://localhost:8080/

Write some ToDo description and click Add button or Press Enter

Not Done items can be marked as Done by clicking Complete button
Done item can be re opened by clicking Pending button
Any ToDo item can be deleted by clicking on its Delete button
